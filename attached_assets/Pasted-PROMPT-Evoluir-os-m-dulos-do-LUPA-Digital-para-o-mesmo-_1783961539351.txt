PROMPT — Evoluir os módulos do LUPA Digital para o mesmo nível do módulo Editais

O módulo Editais é atualmente o módulo mais completo do sistema e deve servir como referência arquitetural, visual e funcional para todos os demais módulos do LUPA Digital.

Objetivo

Não criar novos módulos.

Não alterar o backend do módulo Editais.

Não modificar funcionalidades que já estão funcionando.

O objetivo é elevar os módulos restantes ao mesmo nível de maturidade do Editais.

Antes de implementar

Primeiro faça uma análise completa do módulo Editais e identifique:

fluxo da interface;
componentes reutilizados;
arquitetura React;
comunicação com o backend;
organização dos resultados;
estados de loading;
tratamento de erros;
histórico;
persistência;
integração com AIService;
padrões de layout;
reutilização de componentes.

O módulo Editais será o padrão para todo o restante do sistema.

Experiência do usuário

Todos os módulos devem transmitir a sensação de fazer parte do mesmo produto.

Mesmo padrão de:

botões;

Integração com IA

Nenhum módulo deve chamar diretamente a OpenAI.

Toda chamada deve passar pelo AIService, mantendo:

preservação semântica;
mediação linguística;
linguagem simples;
transparência;
registro de alertas;
logging;
rastreabilidade.
